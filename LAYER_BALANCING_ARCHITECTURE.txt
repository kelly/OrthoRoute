VECTORIZATION EXAMPLE
=====================

Example: Apply bias to Layer 30 (bias = +0.263, meaning hot)

    layer_mask = (self._horizontal_edge_layers == 30)
    # Result: Boolean array [False, False, True, True, False, ...]
    #         Shape: [52,000,000]
    #         True count: ~3,200,000 (edges on Layer 30)

    # Apply bias factor
    bias_factor = 1.0 + 0.263 = 1.263
    total_cost[layer_mask] *= 1.263
    # Result: All ~3.2M edges on Layer 30 get 26.3% cost increase

    # NumPy does this in C, very fast (~10ms per layer)

Why This is Fast:
- Boolean indexing is optimized in NumPy (C backend)
- Vectorized multiply (no Python loops over edges)
- Only loop over 16 layers (not 52M edges)
- Skip layers with negligible bias (80% of layers)
